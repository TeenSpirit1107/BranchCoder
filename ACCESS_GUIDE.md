# 如何访问 AI Chat 扩展

## 🎯 方法 1: 通过活动栏图标（最简单）

启动调试后，在**左侧活动栏**（最左边的垂直工具栏）中，你会看到一个新的图标：
- **💬 聊天图标**（comment-discussion 图标）
- 点击这个图标，就会在侧边栏打开 "AI Chat" 视图
- 在视图里面有一个 "Chat" 面板，这就是聊天界面

## 方法 2: 通过命令面板

1. 按 `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (Mac) 打开命令面板
2. 输入 `AI Chat: Open AI Chat` 并选择
3. 这会自动打开 AI Chat 视图容器

## 如果仍然看不到

1. **确保扩展已正确编译**：
   ```bash
   npm run compile
   ```

2. **检查是否有编译错误**：查看 VS Code 的"问题"面板（Problems）

3. **重新加载调试窗口**：
   - 按 `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (Mac)
   - 输入 `Developer: Reload Window` 并执行

4. **检查调试控制台**：查看是否有错误信息（在调试面板的"调试控制台"标签）

5. **确保扩展已激活**：在调试控制台中应该看到扩展的激活信息

## 视图位置说明

扩展创建了一个独立的视图容器，会在活动栏显示一个图标。如果看不到图标：
- 确保你已经启动了调试（F5）
- 检查活动栏是否显示了所有图标（有些图标可能被隐藏）
